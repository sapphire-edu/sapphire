Everything you need to succeed

Sapphire isn't just a spreadsheet. It's a complete academic command center designed for modern students.

Real-time Performance

See weighted average, top subject, recent assessment, and upcoming work from one dashboard view.

Weighted Analytics Views

Analyze weighted trend, performance highlights, subject score lines, and detailed assessment breakdowns.

Calendar with Type Filters

Plan upcoming work in a dedicated calendar with assessment, homework, and combined filtering.

Sentral Timetable Import

Upload Sentral `.ics` exports to build Week A / Week B timetable views and track classes day by day.

Privacy, Encryption, and Compliance

Account-synced data follows industry security standards, including encrypted string attributes with AES-128-GCM encryption at rest, aligned with SOC 2 Type I.
